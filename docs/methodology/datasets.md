## Example Dataset Entry

### TCGA-SKCM

- **Source**: The Cancer Genome Atlas - Skin Cutaneous Melanoma
- **Description**: Histopathology images for melanoma classification
- **Preprocessing**: Normalized, tiled into 512x512 patches
- **Splits**: 70% train, 15% validation, 15% test
