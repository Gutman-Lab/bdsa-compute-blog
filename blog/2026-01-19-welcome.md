Welcome to the **BDSA Compute Blog**! This site will track experiment results, benchmarks, and updates for the BDSA open source image analysis platform.

## What to Expect

This blog will feature:

- **Release notes** for BDSA versions
- **Experiment highlights** - notable results and findings
- **Methodological updates** - changes to pipelines and evaluation
- **Progress summaries** - periodic updates on project development

## Experiment Tracking

All experiment results are tracked in the [Experiments](/docs/experiments) section, which is automatically generated from structured JSON outputs. This ensures reproducibility and easy comparison across runs.

Stay tuned for updates!
